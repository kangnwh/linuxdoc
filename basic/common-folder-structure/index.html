    Folder structure

    Some common folders & files in Linux will be introduced in this chapter

    /etc

    • /etc/passwd: Stores the user account information. This file contains several columns separated by symbol : For example : root:x:0:0:root:/root:/bin/zsh
    root x 0 0 root /root /bin/zsh
    user_id password, always shows x user sequence id main group sequence id full name of user home dir default shell type
    • /etc/group

    It's just like /etc/passwd but this file contains the information of gorups. Check here for detail.

    /bin & /sbin

    • /bin: The executable applications(commands) that can be accessed from any users.
    • /sbin: The executable applications(commands) that can only be accessed from specific users such as root

    /home

    Default home directory for all users except root.

    /root

    Default home dir for root

    /tmp

    A folder that stores temporary files. Files in this folder will be removed at a specific frequency configured in Linux system.
